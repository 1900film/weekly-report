構造的発見

HIGH日とLOW日の差はIMP量ではなく、CVR変動に起因する

CRITICAL FINDING

IMP量は安定 — CVR変動が全て

HIGH日の平均IMP: 4,857 / LOW日: 4,967。差はわずか2%。
つまりCVR低下日は「配信が減った」のではなく「同じ配信量でCVが取れなかった」。
原因はAG構造と時間帯のCV確率変動にある。

GOLDEN PATTERN

6-7時 早朝帯がCVR最大の分岐点

HIGH日 6時: CVR 4.35% / LOW日 6時: CVR 1.15%（3.8倍差）
HIGH日 7時: CVR 3.07% / LOW日 7時: CVR 0.00%（完全消失）
早朝の検索ユーザーの「転換力」がその日全体のCVRを左右している。

SECTION 06

構造的結論とアクション

データから導かれる優先アクション

CVR変動の正体は「小AG x 時間帯」の確率揺れ

602__老眼・605__症状は日間CV 0-1件のため、1日単位でCVR 0%↔4%と激しく振れる。
501__比較検討は安定しているが、12-14時と15-17時のLOW日CVR消失が大きな損失源。
603__視力はHIGH日に高効率(CPA ¥4,848)だが振幅が最大。

  • P1 501 早朝ブースト: 06-08時のCVR差が最大（2.93% vs 0.74%）。HIGH日パターンを再現するため、早朝の入札調整を+20%に引き上げ
  • P1 501 昼帯テコ入れ: 12-14時のLOW日CVR 0%は広告文またはLPの問題。昼休みユーザー向けの訴求テスト実施
  • P2 602/605 統合検討: 日間CV 0-1件でSmart Biddingの学習が不安定。601と統合してデータ量を確保するか、手動入札に切替
  • P2 603 夕方・夜間強化: 15-17時 CVR 5.62%、18-20時 CVR 4.00%はHIGH日最高効率。この時間帯の入札を積極化
  • P2 601 深夜帯活用: 21-23時のCVR 3.68%(HIGH日)はCPA ¥3,262で最高効率。深夜帯の予算配分を増加
  • P3 18時台の構造的問題: HIGH日でもCVR 0.32%と低迷。全AGで18時台は「転換力の谷」。入札抑制または除外を検討
  • P3 曜日パターン検証: LOW日が特定曜日に集中するか追加分析。週末(土日)のCVRパターンが平日と異なる可能性
